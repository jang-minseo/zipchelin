## **집슐랭(zipchelin)이란?**
집슐랭은 <b>요리 레시피</b>를 한 곳에 모아 <b>이용자</b>가 쉽게 요리 레시피를 검색, 공유를 할 수 있는 사이트입니다. 
<br><br>
